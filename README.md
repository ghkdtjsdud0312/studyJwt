# 스프링 부트 + mysql + Spring Data JPA를 활용한 회원 프로젝트 입니다.

# 개발 환경 소개
1. 개발 환경
    - Open JDK 11
    - IntelliJ IDEA Community
    - MySQL Server 8.0
    - SpringBoot 2.6.13
2. Library of Framework
    - Spring Data JPA
    - Thymeleaf
    - lombok
    - gradle

# 주요 기능
- 회원가입
- 로그인
- 회원목록
- 회원조회
- 회원정보 수정
- 회원삭제
- 로그아웃

## 추가 기능

- 기존 MemberProject에 추가 하는 기능
